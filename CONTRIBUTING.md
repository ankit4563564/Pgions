# Contributing to Pgions

First off, thank you for considering contributing to Pgions! It's people like you that make Pgions such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) page. 

If you want to contribute code:
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Code Style

- We use standard JavaScript/Next.js conventions.
- Custom CSS should be placed in `.module.css` files using Vanilla CSS. Try to utilize CSS variables defined in `/app/globals.css` to maintain the Glassmorphic Design System.

Thank you!
